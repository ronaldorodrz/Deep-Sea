# Deep-Sea
Eu desenvolvi este jogo enquanto estava aprendendo programação com Python. O curso foi realizado de forma online pela instituição Universidade Estadual de Feira de Santana (UEFS), de maneira totalmente gratuita, no ano de 2024. O projeto foi baseado em um jogo de tabuleiro chamado Deep Sea, no qual precisávamos seguir as orientações e requisitos solicitados para a criação do sistema.
