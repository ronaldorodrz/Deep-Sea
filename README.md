# Deep-Sea
Este projeto foi desenvolvido durante meu aprendizado em Python.
O curso foi realizado de forma online pela Universidade Estadual de Feira de Santana (UEFS), de maneira totalmente gratuita, no ano de 2024.

O jogo é baseado no tabuleiro Deep Sea, seguindo as orientações e requisitos propostos para a criação do sistema.

Por se tratar de um projeto inicial, o foco principal foi aplicar os conceitos aprendidos e garantir o funcionamento da lógica do jogo, priorizando a compreensão prática da programação.
